### Usage:

```bash
   path/usr/$: ./better_git_log.sh -l 5 -s 10
```

#### 
```bash
   OPTIONS:
         line amount  -l N      [ N must be greater than 4 lines]
         seconds amount  -s N   [ N must be greater than 4 seconds ]
```
